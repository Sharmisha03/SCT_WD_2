# ⏱ Stopwatch Web Application

## 📌 Project Description
This project is a responsive and interactive stopwatch web application developed as part of my Web Development Internship. The application is designed to accurately measure and record time intervals with a clean and user-friendly interface.

It includes essential stopwatch functionalities such as start, pause, reset, and lap recording, along with interval tracking between laps. The interface follows modern UI design principles using a dark theme, glassmorphism, and glow effects to enhance visual appeal and usability.

---

## 🚀 Features
- Start, pause, and reset functionality  
- Accurate time tracking with milliseconds precision  
- Lap recording with automatic numbering  
- Interval time calculation between consecutive laps  
- Real-time display updates  
- Scrollable lap history  
- Responsive and user-friendly design  
- Modern UI with glass effect and neon glow styling  

---

## 🛠️ Technologies Used
- HTML5  
- CSS3  
- JavaScript (Vanilla JS)
